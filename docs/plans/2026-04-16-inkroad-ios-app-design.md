# INKROAD iOS App - High-Level Design

## 1. 개요 (Overview)
기존 구축된 INKROAD 웹소설 스토어(HTML/CSS/JS 기반)를 바탕으로, 동일한 서비스 경험을 제공하는 iOS 겸용 크로스플랫폼 모바일 앱을 구축합니다.

## 2. 기술 스택 (Technology Stack)
- **프레임워크:** React Native + Expo (옵션 A)
- **선정 이유:** 기존 웹 프론트엔드 환경에서 전환하기 가장 빠르며, 복잡한 네이티브 환경(Xcode 등) 세팅 없이 'Expo Go'를 통해 즉각적인 모바일 기기 테스트가 가능합니다. 초보자 친화적이고 배포가 직관적입니다.
- **백엔드:** Supabase

## 3. 핵심 아키텍처 및 화면 구조 (App Structure)
기존 모바일 웹 UI를 바탕으로 네이티브 탭 네비게이션(Tab Navigation) 구조를 채택합니다.

1. **홈 (Home 탭):** 
   - 메인 히어로 배너 (추천작)
   - 장르별 퀵 메뉴
   - 인기 작품 및 최근 업데이트 가로 스크롤 (FlatList 활용)
2. **탐색 (Search 탭):** 
   - 검색창 및 태그 기반 필터링
3. **서재 (My Library 탭):** 
   - 사용자가 찜하거나 열람 중인 소설 리스트
4. **마이 프로필 (MY 탭):** 
   - 로그인 / 내 정보 관리 / 코인 결제 (추후 인앱 결제 연동 고려)

## 4. 데이터 연동 (Data Integration)
- **Supabase SDK 연동:** 웹사이트에서 사용 중인 Supabase 프로젝트를 그대로 연결합니다.
- **데이터 활용:** 현재 Supabase에 들어있는 더미(Dummy) 데이터를 활용하여 UI 컴포넌트와 리스트가 정상적으로 렌더링되는지 개발/테스트합니다. (개발 단계에서 더미 데이터 사용은 매우 권장되는 방식입니다.)
- **계정 통합:** 앱과 웹이 동일한 DB를 바라보므로 사용자 계정 정보 역시 100% 동기화됩니다.

## 5. UI/UX 디자인 원칙
- 모바일에 최적화된 부드러운 스크롤과 터치 피드백을 제공합니다.
- 웹사이트의 CSS 기반 테마(색상, 폰트)를 React Native StyleSheet 객체로 이식하여 통일된 브랜드 경험을 유지합니다.
