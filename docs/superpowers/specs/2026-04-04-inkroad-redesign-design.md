# 잉크로드 PC 리디자인 — Clean Light Store

## 개요

잉크로드는 웹소설 글로벌 마켓플레이스다. 번역을 통한 소설 수출입 + Steam식 주기적 할인이 핵심 비즈니스 모델이다.

현재 디자인(다크 글래스모피즘, 과도한 그라데이션, 44px border-radius, 분산된 인라인 스타일)을 전면 리디자인한다. 방향은 **Clean Light Store** — 밝은 웜 화이트 배경에 표지 이미지가 주인공이 되는 깔끔한 스토어프론트.

**범위:** 독자 핵심 동선 4개 페이지 우선 (홈 → 탐색 → 작품 상세 → 뷰어). 나머지 페이지(인증, 결제, 크리에이터 대시보드, 업로드 등)는 이 디자인 시스템을 기반으로 후속 작업.

**결제 모델:** 편당(회차당) 결제. 편당 가격은 플랫폼 전체에서 고정 단가(예: 300원). 할인 이벤트 시 할인율만 변동. 책 전체 가격은 존재하지 않는다.

## 디자인 시스템

### 색상

| 토큰 | 값 | 용도 |
|------|------|------|
| `--bg` | `#f7f5f2` | 페이지 배경 (웜 화이트) |
| `--surface` | `#ffffff` | 카드, 패널, 네비게이션 |
| `--surface-muted` | `#f0eeeb` | 검색창, 비활성 필터, 입력 필드 |
| `--text` | `#1a1a1a` | 제목, 주요 텍스트 |
| `--text-secondary` | `#444444` | 본문, 설명 텍스트 |
| `--text-muted` | `#999999` | 메타 정보, 보조 텍스트 |
| `--border` | `#eeeeee` | 구분선, 카드 테두리 |
| `--accent-sale` | `#e85d3a` | 할인, 세일 CTA, 가격 강조 |
| `--accent-free` | `#2d8a4e` | 무료 뱃지, 완료/성공 상태 |
| `--dark` | `#1a1a1a` | 히어로 배너, 장르 카드, 주요 버튼 |

### 피해야 할 색상

- 채도 높은 보라 단색
- 보라-핑크-블루 그라데이션
- 검정 + 네온 퍼플
- 민트 + 보라 조합
- 밝은 하늘색 메인

이런 조합은 AI 챗봇/이미지 생성 툴/스타트업 랜딩 느낌을 준다.

### 타이포그래피

- **폰트:** Pretendard(한글) + Inter(영문). 세리프 사용 안 함.
- **제목:** 20-32px, font-weight 700-800, letter-spacing -0.03em
- **섹션 제목:** 16-20px, font-weight 700
- **본문:** 14px, font-weight 400, color #444, line-height 1.7
- **메타:** 12-13px, color #999
- **뷰어 본문:** 16px, line-height 2.0, color #333, text-indent 1em

### 레이아웃

- **max-width:** 960px (콘텐츠 영역)
- **border-radius:** 8-12px (기존 32-44px에서 대폭 축소)
- **그림자:** `0 1px 3px rgba(0,0,0,0.04)` ~ `0 1px 4px rgba(0,0,0,0.05)` (최소한)
- **카드 그리드:** 4열 기본 (기존 다양한 열 수에서 통일)
- **간격:** 패딩 14-18px(카드 내부), 갭 16px(그리드), 섹션 간 36px

### 컴포넌트

**버튼:**
- Primary: `background: #1a1a1a; color: #fff` — 읽기, 주요 액션
- Sale CTA: `background: #e85d3a; color: #fff` — 할인 구매
- Secondary: `background: #fff; border: 1px solid #ddd; color: #1a1a1a` — 위시리스트, 보조
- Ghost: `background: #f0eeeb; color: #666` — 필터, 비활성

**뱃지:**
- 할인: `background: #e85d3a; color: #fff` — "-45%", "이번 주 할인"
- 무료: `background: #2d8a4e; color: #fff` — "30화 무료", "무료"
- 신작: `background: #1a1a1a; color: #fff` — "NEW"
- 메타: `background: #f0eeeb; color: #666` — 장르, 출처, 번역 방향

**노블 카드 (그리드용):**
- 흰색 배경, border-radius 12px, 미세한 그림자
- 상단: 표지 이미지 (aspect-ratio 0.72), 할인/무료 뱃지 오버레이
- 하단: 제목(14px, bold), 장르·출처(12px, #999), 편당 가격

**노블 카드 (리스트용, 탐색 페이지):**
- 흰색 배경, 가로 배치: 표지(100px) | 제목+메타+설명+태그 | 편당 가격+CTA
- 한 줄에 하나씩, 비교가 쉬운 구조

**네비게이션 바:**
- 흰색 배경, 하단 1px border
- 좌: 로고(INKROAD, 20px bold) + 메뉴(스토어/탐색/내 서재/세일)
- 우: 검색창 + 언어 전환(KR ▾) + 로그인 버튼
- "세일" 메뉴는 #e85d3a 색상으로 강조

## 페이지별 레이아웃

### 1. 홈페이지 (`homepage_pc.html`)

위에서 아래 스크롤 순서:

**① 네비게이션 바**
- sticky, 흰색 배경

**② 히어로 배너 (다크 섹션)**
- 배경 #1a1a1a, 전체 너비
- 추천 작품 1개: 뱃지(할인/번역 방향/장르) + 제목 + 번역명 + 설명 + CTA(무료로 읽기, 위시리스트) + 메타(총 화수, 평점, 할인 정보)
- 오른쪽에 표지 이미지 (180px)

**③ 세일 배너**
- 흰색 카드, 현재 이벤트명 + 작품 수 + 할인율 + 카운트다운 타이머 + "전체 보기" 링크

**④ "지금 할인 중" 섹션**
- 섹션 제목(20px) + "전체 보기" 링크
- 4열 노블 카드 그리드
- 각 카드: 표지 + 할인 뱃지 + 제목 + 장르·출처 + `편당 300원 → 편당 165원` (고정 단가 기준)

**⑤ "무료 연재" 섹션**
- 동일 4열 그리드
- 그린 뱃지: "30화 무료", "20화 무료" 등
- 가격 대신 평점 표시

**⑥ "장르별 탐색" 섹션**
- 4열 다크 카드: 판타지, 로맨스, 회귀, 번역작 등
- 각 카드에 작품 수 표시

**⑦ 푸터**
- 심플. 저작권 + 이용약관/개인정보/고객센터

### 2. 탐색 페이지 (`search_pc.html`)

**① 네비게이션 바** (동일)

**② 검색 바**
- 네비 아래 전체 너비 영역, 흰색 배경
- 통합 검색: 작품명, 작가, 태그

**③ 2컬럼 레이아웃: 필터(220px) + 결과**

필터 사이드바 (sticky):
- 장르: 판타지, 로맨스, 회귀, 아카데미, 하렘, 피폐 등 (칩 토글)
- 출처: 한국, 일본, 중국, 영미
- 상태: 연재중, 완결, 무료 구간 있음, 할인 중
- 정렬: 인기순, 최신순, 평점순, 할인율순

결과 영역:
- 상단: 결과 수 표시 ("1,284개 작품")
- 리스트형 카드: 표지(100px) | 제목+메타+설명+태그 | 편당 가격+CTA
- 할인 작품: 취소선 원가 + 할인가 (고정 단가 기준)
- 비할인 작품: 정가만 표시

### 3. 작품 상세 페이지 (`novel_detail_pc.html`)

**① 네비게이션 바** (동일)

**② 작품 정보 영역**
- 2컬럼: 표지(200px) + 정보
- 표지 아래: "1화 읽기" 버튼 + 하트 버튼
- 정보:
  - 뱃지 (무료 구간, 할인 정보)
  - 제목 (28px, bold)
  - 번역명
  - 메타 그리드: 작가 / 상태(완결·245화) / 평점(★ 4.9) / 출처(한국 → 영어)
  - 장르 태그 (칩)
  - 시놉시스 (본문 텍스트)

**③ 가격 박스**
- 흰색 카드, 정보 영역 하단
- 좌: "유료 회차 가격" 라벨 + 취소선 원가(편당 300원) + 할인가(편당 165원) + 유료 회차 수 + 할인 남은 시간
- 우: CTA 버튼 ("할인가로 이어 읽기")

**④ 회차 목록**
- 상단: "회차 목록" 제목 + 필터 탭 (전체/무료/유료)
- 리스트: 회차 번호 + 제목 + 무료·잠금 상태
  - 무료 회차: 그린 "무료" 뱃지
  - 유료 회차: 🔒 + 가격 (할인 시 취소선 + 할인가)
  - 무료 구간 축약: "4화 ~ 30화 (무료)" 한 줄로

**⑤ 비슷한 작품**
- 4열 노블 카드 그리드

### 4. 뷰어 페이지 (`novel_viewer_pc.html`)

**① 미니멀 네비**
- "← 작품으로" + 작품명 + 현재 회차
- 우측: 하트, 설정

**② 읽기 진행률 바**
- 네비 바로 아래, 3px 높이
- 스크롤 위치에 따라 #1a1a1a로 채워짐

**③ 본문 영역**
- max-width 680px, 중앙 정렬
- 챕터 헤더: 권 정보 + 회차 제목 + 무료/유료 표시 (중앙 정렬, 하단 구분선)
- 본문: 16px, line-height 2.0, color #333, text-indent 1em
- 배경: #f7f5f2 (페이지 배경과 동일, 별도 카드 없이 자연스럽게)

**④ 하단 회차 이동 바**
- 하단 고정(sticky bottom), 흰색 배경
- 3컬럼: 이전화 | 목차 + 현재/전체 | 다음화
- 다음 화가 무료면 그린 뱃지, 유료면 가격 표시

**⑤ 페이월 (유료 회차 도달 시)**
- 본문 스크롤 끝에 자연스럽게 등장
- 흰색 카드, 중앙 정렬
- "무료 구간이 끝났습니다" + 회차 제목
- 정가/할인가 비교 + "이 화 구매하기" CTA + 할인 남은 시간

## CSS 구조 변경

현재 상태:
- `assets/styles.css` — 공용 스타일 (모바일 + PC 혼용)
- `assets/pc.css` — PC 전용
- 일부 페이지에 인라인 `<style>` 블록 (auth_pc, creator_dashboard, episode_upload, novel_upload)

리디자인 후:
- `assets/styles.css` — 디자인 토큰(CSS 변수) + 리셋 + 공용 컴포넌트(버튼, 뱃지, 카드, 네비)
- `assets/pc.css` — PC 전용 레이아웃 (그리드, 여백)
- 인라인 `<style>` 블록은 모두 `pc.css`로 병합
