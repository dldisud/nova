# InkRoad 프로젝트 작업 계획 (Plan)

## 📌 현재 프로젝트 진행 상태

**✅ 완료된 작업 (Clean Light Store 리디자인)**
- **PC 리더 코어 4개**: 홈 (`homepage_pc.html`), 검색 (`search_pc.html`), 상세 (`novel_detail_pc.html`), 뷰어 (`novel_viewer_pc.html`)
- **PC 마이페이지 및 대시보드**: 내 서재 (`my_library_pc.html`), 로그인 (`auth_pc.html`), 작품 관리 (`creator_dashboard_pc.html`), 통계 (`author_dashboard_pc.html`)
- **PC 크리에이터 업로드**: 작품 등록 (`novel_upload_pc.html`), 회차 발행 (`episode_upload_pc.html`)

---

## 🔲 앞으로 해야 할 작업 (우선순위 순)

| 우선순위 | 영역 | 상세 내용 |
|:---:|---|---|
| **1** | **반응형 통합 (모바일/태블릿)** | PC/모바일 HTML을 하나로 합치고 미디어 쿼리(768~1024px 등) 기반 반응형 구축 |
| **2** | **결제 연동** | `payment_pc.html` 리디자인 및 Stripe 실결제 플로우 연동 |
| **3** | **소셜 기능** | 댓글, 리뷰, 알림 기능 프론트엔드 및 백엔드 연동 |
| **4** | **SEO 최적화** | Open Graph(OG) 태그, 구조화 데이터 등 검색엔진 최적화 |
| **5** | **실서버 배포** | Vercel 또는 Netlify에 최종 배포 및 프로덕션 환경 셋업 |

### 💡 다음 스텝 제안
PC 버전의 전반적인 리디자인과 UI 구현이 완료되었으므로, 가장 먼저 **1순위인 모바일 및 반응형(미디어 쿼리) 통합 작업**에 들어가는 것을 추천합니다. 이를 통해 완전히 반응형으로 동작하는 서비스를 만들고 이후 결제/SEO 등을 덧붙이는 방향이 효율적입니다.
