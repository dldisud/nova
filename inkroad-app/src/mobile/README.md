## Mobile screen map

- `homepage.html` -> `screens/HomeScreen.tsx`
- `search.html` -> `screens/SearchScreen.tsx`
- `my_library.html` -> `screens/LibraryScreen.tsx`
- `profile_settings.html` -> `screens/ProfileScreen.tsx`
- `auth.html` -> `screens/AuthScreen.tsx`
- `novel_detail.html` -> `screens/NovelDetailScreen.tsx`
- `novel_viewer.html` -> `screens/NovelViewerScreen.tsx`
- `payment.html` -> `screens/PaymentScreen.tsx`

Shared mock content lives in `data/mockInkroad.ts`.
Shared UI pieces live in `components/`.
